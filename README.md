# Rustic Barber — Barbería

Sistema de reserva de citas para barbería.
Hecho con HTML5, CSS3 y JavaScript Vanilla.

---

# Estructura del proyecto

```
barberia/
├── index.html        ← Toda la estructura de la página
├── css/
│   └── styles.css    ← Todos los estilos
├── js/
│   └── app.js        ← Toda la interactividad
└── README.md         ← Este archivo
```

---

# Paleta de colores rústica

| Variable         | Color      | Uso                        |
|------------------|------------|----------------------------|
| `--crema`        | #F5EFE0    | Fondo principal            |
| `--pergamino`    | #EAE0CC    | Fondo secciones alternas   |
| `--dorado`       | #B8882A    | Color de acento principal  |
| `--dorado-claro` | #D4A843    | Hover, highlights          |
| `--cafe-oscuro`  | #2E1F0E    | Fondo oscuro, textos       |
| `--negro`        | #1A1208    | Header, footer             |


# Funcionalidades incluidas

- Menú hamburguesa para móvil
- Scroll suave con offset del header fijo
- Link activo en la navegación al hacer scroll
- Diseño responsivo (mobile first)

---

# Cómo usar

1. Descarga la carpeta
3. Abre `index.html` en tu navegador
4. ¡Listo!
